# Blog — checklist

## P0

- [ ] One `<h1>` per page.
- [ ] Prose container `max-w-[70ch]` or `max-w-3xl`.
- [ ] Line-height ≥ 1.5 on body.
- [ ] Mobile-safe at 375px (long words wrap, code blocks scroll).
- [ ] Editorial theme installed.

## P1

- [ ] `<time datetime="ISO">` wraps every date display.
- [ ] Reading-time estimate on posts.
- [ ] Anchor links on h2/h3 in posts (hash on hover).
- [ ] Open-graph metadata via `+page.server.ts`.
