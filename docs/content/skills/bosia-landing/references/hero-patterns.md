# Hero patterns

Three patterns; pick by content.

## 1. Centered text (default)

- Headline + subhead + CTA row, centered.
- No imagery.
- Fits: products where the value prop is conceptual; design-led brands.

Use when you can't ship a screenshot that's immediately legible at small sizes.

## 2. Asymmetric — text + visual

- Left: headline + subhead + CTA.
- Right: product screenshot, illustration, or video.
- Two-column on desktop, stacked on mobile.

Use when the product _is_ its UI (dashboards, editors).

## 3. Full-bleed image with overlaid text

- Background image fills the hero.
- Text overlaid with a contrast scrim.
- Works only when the image is genuinely strong.

Use when the brand is image-led (consumer, lifestyle). Rarely the right call for B2B tools.

## Anti-patterns

- "Hero carousel" — every slide hides the others.
- Two competing CTAs the same size.
- Lorem-style headlines that don't say what the product does.
