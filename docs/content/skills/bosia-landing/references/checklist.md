# Landing — review checklist

## P0

- [ ] One `<h1>`, above the fold.
- [ ] One primary CTA above the fold; one secondary at most.
- [ ] Semantic tokens only (no raw colors).
- [ ] Mobile renders without horizontal scroll at 375px.
- [ ] Feature cards installed via `bosia add`, not pasted.

## P1

- [ ] Headline ≤ 9 words.
- [ ] Subhead ≤ 24 words, contains one concrete benefit.
- [ ] Feature card titles are verb-led.
- [ ] Footer uses `<footer>` landmark.
- [ ] Images have descriptive `alt`.

## Copy smell tests

- Replace any "we" or "our" with the product noun — does it still make sense?
- If you delete the subhead, does the headline still communicate value?
- If you delete every adjective, does the page still sell?

If yes to all three, the copy is doing its job.
