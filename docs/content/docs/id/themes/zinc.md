```bash
bun x bosia@latest add theme zinc
```

Palet biru-abu sejuk terinspirasi zinc shadcn. Estetika bersih dan minimal, sempurna untuk developer tools, dashboard, dan aplikasi teknis. Tumpukan font sistem — tanpa biaya web font.

## Tokens

- Primary biru-abu sejuk: `240 5.9% 10%`
- Mode terang/gelap kontras tinggi
- `--radius: 0.5rem`
- `--font-display` dan `--font-body` keduanya system-ui (tanpa web font)
- Nilai HSL agar modifier opasitas Tailwind berfungsi: `bg-primary/80`, dll.

Sumber: `registry/themes/zinc/tokens.css`.
