```bash
bun x bosia@latest add theme forest
```

Palet hijau earthy dengan aksen amber hangat, sempurna untuk aplikasi alam, keberlanjutan, dan kesehatan. Primary hijau pekat menyampaikan pertumbuhan dan stabilitas; aksen amber menambah kehangatan. Tumpukan font sistem — tanpa biaya web font.

## Tokens

- Primary hijau forest: `142 72% 29%`
- Aksen amber hangat: `38 92% 48%`
- Hijau dan nada earthy dengan nuansa organik
- `--radius: 0.375rem` (sedikit membulat)
- `--font-display` dan `--font-body` keduanya system-ui (tanpa web font)
- Nilai HSL agar modifier opasitas Tailwind berfungsi: `bg-primary/80`, dll.

Sumber: `registry/themes/forest/tokens.css`.
