---

```bash
bun x bosia@latest add theme sage
```

Hijau sage yang redup dan tenang — kalem dan ramah konten. Radius kecil, bayangan ambient lembut. Display dengan **Bricolage Grotesque**, body dengan **Plus Jakarta Sans**.

## Tokens

- `--radius: 0.625rem` yang kecil
- Brand dipetakan ke `--primary` (hijau redup) — gunakan `bg-primary` / `text-primary`
- Elevasi bernuansa-foreground kalem: `shadow-sm` / `shadow-md` / `shadow-lg`
- `--font-mono: "JetBrains Mono"` untuk angka
- Terang + gelap lewat `:root` / `.dark`

Sumber: `registry/themes/sage/tokens.css`.
