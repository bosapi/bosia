```bash
bun x bosia@latest add theme sunset
```

Primary oranye hangat dengan aksen pink untuk aplikasi konsumen yang energik, landing page, dan situs marketing. Tumpukan font sistem — tanpa biaya web font.

## Tokens

- Primary oranye cerah: `20 95% 53%`
- Aksen Pink-340 untuk sorotan dan CTA sekunder
- `--radius: 0.625rem` (modern, sedikit membulat)
- `--font-display` dan `--font-body` keduanya system-ui (tanpa web font)
- Nilai HSL agar modifier opasitas Tailwind berfungsi: `bg-primary/80`, dll.

Sumber: `registry/themes/sunset/tokens.css`.
