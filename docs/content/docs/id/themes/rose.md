```bash
bun x bosia@latest add theme rose
```

Palet rose cerah terinspirasi tema rose shadcn. Sempurna untuk aplikasi kreatif, fashion, kecantikan, dan brand gaya hidup. Primary berani yang energik dan menarik perhatian dengan kontras sangat baik. Tumpukan font sistem — tanpa biaya web font.

## Tokens

- Primary rose berani: `346.8 77.2% 49.8%`
- Putih bersih dengan aksen rose
- `--radius: 0.5rem`
- `--font-display` dan `--font-body` keduanya system-ui (tanpa web font)
- Nilai HSL agar modifier opasitas Tailwind berfungsi: `bg-primary/80`, dll.

Sumber: `registry/themes/rose/tokens.css`.
