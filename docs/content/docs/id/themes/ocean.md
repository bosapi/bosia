```bash
bun x bosia@latest add theme ocean
```

Palet navy dan teal profesional untuk dashboard SaaS, aplikasi data, dan perangkat lunak korporat. Primary navy pekat menyampaikan kepercayaan; aksen teal menambah nuansa modern berorientasi teknologi. Tumpukan font sistem — tanpa biaya web font.

## Tokens

- Primary teal pekat: `199 89% 40%`
- Aksen teal cerah: `175 84% 32%`
- Navy, biru, dan teal dengan kontras tinggi
- `--radius: 0.5rem`
- `--font-display` dan `--font-body` keduanya system-ui (tanpa web font)
- Nilai HSL agar modifier opasitas Tailwind berfungsi: `bg-primary/80`, dll.

Sumber: `registry/themes/ocean/tokens.css`.
