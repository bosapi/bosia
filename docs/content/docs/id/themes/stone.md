```bash
bun x bosia@latest add theme stone
```

Palet abu hangat terinspirasi tema stone shadcn. Estetika nyaman menyerupai kertas, sempurna untuk situs konten, platform penerbitan, dan editor dokumen. Tumpukan font sistem — tanpa biaya web font.

## Tokens

- Primary cokelat hangat: `24 9.8% 10%`
- Palet abu hangat dengan nuansa earthy
- `--radius: 0.5rem`
- `--font-display` dan `--font-body` keduanya system-ui (tanpa web font)
- Nilai HSL agar modifier opasitas Tailwind berfungsi: `bg-primary/80`, dll.

Sumber: `registry/themes/stone/tokens.css`.
