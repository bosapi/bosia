---
title: Tema Amber
description: Primary amber + aksen kuning. Estetika hospitality dan makanan yang nyaman.
---

```bash
bun x bosia@latest add theme amber
```

Primary amber hangat dengan aksen kuning dan radius yang lega. Palet nyaman untuk aplikasi hospitality, makanan, dan gaya hidup yang menginginkan nuansa lembut dan ramah. Tumpukan font sistem — tanpa biaya web font.

## Tokens

- Primary amber: `38 92% 50%`
- Aksen Yellow-45 untuk sorotan dan badge
- `--radius: 1rem` (sudut lembut, empuk)
- `--font-display` dan `--font-body` keduanya system-ui (tanpa web font)
- Nilai HSL agar modifier opasitas Tailwind berfungsi: `bg-primary/80`, dll.

Sumber: `registry/themes/amber/tokens.css`.
