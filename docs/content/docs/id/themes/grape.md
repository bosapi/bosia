---
title: Tema Grape
description: Ungu lilac gelap dengan kilau aksen lembut.
---

```bash
bun x bosia@latest add theme grape
```

Ungu lilac gelap dengan **kilau** aksen lembut. Display dengan **Bricolage Grotesque**, body dengan **Plus Jakarta Sans**. Varian terang disediakan untuk `:root` sehingga toggle Terang/Gelap tetap berfungsi.

## Tokens

- `--radius: 0.875rem`
- Brand dipetakan ke `--primary` (lilac) — gunakan `bg-primary` / `text-primary`
- Elevasi kilau: `shadow-lg` gelap menambahkan halo `hsl(var(--primary))` lembut
- `--font-mono: "JetBrains Mono"` untuk angka
- Dark-native, dengan varian `:root` terang

Sumber: `registry/themes/grape/tokens.css`.
