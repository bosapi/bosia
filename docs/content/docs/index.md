# Welcome to Bosia

Bosia is a fullstack framework combining **Bun**, **Svelte 5**, and **ElysiaJS** for building fast, modern web applications.

## Quick Start

```bash
bun create bosia my-app
cd my-app
bun run dev
```

## Key Features

- **Server-side rendering** with streaming support
- **File-based routing** with layouts and data loading
- **API routes** with form actions
- **Built-in security** (CSRF, XSS, path traversal protection)
- **Tailwind CSS v4** integration
- **Component registry** for sharing UI components
- **TypeScript** support out of the box
- **Hot module replacement** in development

## Why Bosia?

- **Lightweight**: Built on proven technologies (Bun, Elysia, Svelte)
- **Developer-friendly**: Familiar SvelteKit-inspired API
- **Fast**: Optimized for performance with streaming SSR
- **Secure by default**: Security headers, CSRF protection, input validation
- **Full-stack**: Backend and frontend in one framework
