## In progress — Complete the Indonesian (`id/`) translation

> The docs ship a partial Bahasa Indonesia translation under
> `content/docs/id/`. **136 English pages still lack an `id/` counterpart**,
> so their hreflang alternates point nowhere and the language switcher
> dead-ends. This finishes full parity.
>
> **Out of scope:** Skills (`content/skills/*`) target AI agents, not human
> readers. `reference/changelog` + `reference/roadmap` are auto-generated by
> `sync-docs.ts` from the root `CHANGELOG.md`/`ROADMAP.md` (English-only, no
> `id/` sync) — translating them is a snapshot that drifts stale every
> release, so they're skipped.

**Per-page contract:** mirror the EN frontmatter (translate `title`/
`description` prose, keep keys); translate prose only — keep code blocks, CLI
commands, prop-table identifiers, and API names verbatim; preserve relative
links (the language switcher rewrites the `/id/` prefix).

Phased so each batch is reviewed before the next:
